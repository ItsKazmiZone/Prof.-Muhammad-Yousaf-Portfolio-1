# 📖 About

This portfolio website showcases the academic journey, teaching experience, qualifications, and online tuition services of **Prof. Muhammad Yousaf**.

The project focuses on providing students and parents with a clean, professional, and user-friendly experience while reflecting academic excellence and professionalism.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/ItsKazmiZone/Prof.-Muhammad-Yousaf-Portfolio-1.git
```

Move into the project

```bash
cd Prof.-Muhammad-Yousaf-Portfolio-1
```

Open the website

```text
index.html
```

No installation or build tools are required.
